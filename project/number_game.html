I remeber that there is a game I played since I was little in Taiwan.
I decided to write a small and simple Andrio app for the game.
Basically, when the user start the game,
the app will generate a four-digital ramdon number with repeatition.
The goal of the user is to guess what the four-digital buber is.
Every time when the user entered a number, the app will show a result in the form of #A#B
Where # represents a number from 0 to 4.
A means that the user guessed match the answer and is at the correct spot.
B means that the user guessed match the answer but are in the wring spot.

I remeber that there is a game I played since I was little in Taiwan.
I decided to write a small and simple Andrio app for the game.
Basically, when the user start the game,
the app will generate a four-digital ramdon number without repeatition.
The goal of the user is to guess what the four-digital number is.
Every time when the user enter a number, the app will show a result in the form of #A#B
Where # represents a number from 0 to 4.
An A means that the user has guessed the number correctly and it is at the correct spot.
A B means that the user has the number correctly but it is in the wring spot.
